# TaskFlow â€“ Project Management Web Application

TaskFlow is a modern project management web application designed to provide a clean, efficient, and responsive experience for managing tasks and workflows.  
Built with **React (Vite + TypeScript)** on the frontend and **.NET + Supabase** on the backend, the system emphasizes speed, reliability, and an intuitive interface powered by optimistic UI updates.

## Features

| Feature Category       | Features                                                                                   |
|------------------------|--------------------------------------------------------------------------------------------|
| **Task Management**    | - Create, update, and delete tasks with immediate feedback<br>- Optimistic UI for fast, responsive interactions<br>- Priority, status, and due-date support |
| **User Interface & Experience** | - Clean and minimal layout using shadcn/ui + Tailwind<br>- Fast load times with Vite<br>- Fully responsive design |
| **Project Dashboard**  | - Overview of tasks across projects<br>- Sorting, filtering, and searching<br>- Efficient data handling via RTK Query |
| **Authentication**     | - Supabase authentication (Email/Password)<br>- Secure session handling via backend |
